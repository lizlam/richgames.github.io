Basic Brandub

Brandub is an Irish variant of the ancient Viking game of hnefatafl.

This board is hand-made using 8 inch square plywood that is 1/8 inch thick. The pattern is burned into the wood using the pyrography techique. The center and corner squares are stained for emphasis using 2 layers of coffee. Included is a set of glass gems, 8 clear attackers, 4 black defenders and a red king.

As with many hand-made products, no two boards are alike. There may be minor irregularities that gives each set a unique rustic character.

WARNING: CHOKING HAZARD - This product contains small parts and is not suitable for children.
